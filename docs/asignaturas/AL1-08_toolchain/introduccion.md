# Introducción a AL1-08 · Toolchain (Git, CMake, VS Code)

## ¿Por qué esta asignatura?

Las herramientas de desarrollo profesional son fundamentales para proyectos de audio escalables. Esta asignatura te proporciona:

- **Dominio** de Git para control de versiones colaborativo
- **Conocimiento** de CMake para builds multiplataforma reproducibles
- **Eficiencia** en VS Code con configuraciones avanzadas
- **Habilidad** para depurar y optimizar código C++ profesionalmente

## Prerrequisitos

- Programación básica en C++ (AL1-02 recomendado)
- Familiaridad con línea de comandos
- Sistema operativo: Windows/macOS/Linux

## Estructura del Curso

El curso se divide en **5 bloques temáticos** con **6 capítulos** cada uno:

### Bloque I: Git para Control de Versiones (Caps. 1-6)
Repositorios, commits, branches, merge, rebase, resolución de conflictos, GitHub/GitLab.

### Bloque II: CMake para Build Multiplataforma (Caps. 7-12)
CMakeLists.txt, targets, librerías, find_package, instalación, generadores.

### Bloque III: VS Code Avanzado (Caps. 13-18)
Configuración workspace, extensiones (C++, Python, CMake Tools), tasks.json, launch.json.

### Bloque IV: Depuración y Profiling (Caps. 19-24)
GDB, LLDB, breakpoints, watch expressions, memory inspectors, valgrind, sanitizers.

### Bloque V: Proyecto - Pipeline de Build Completo (Caps. 25-30)
Proyecto C++ con CMake, Git, CI/CD básico (GitHub Actions), tests automatizados.

## Metodología

Cada capítulo incluye:

- **Teoría**: Conceptos de herramientas con ejemplos prácticos
- **Tutoriales paso a paso**: Configuraciones completas ejecutables
- **Best practices**: Convenciones profesionales de la industria
- **Micro-laboratorio**: Ejercicios hands-on con toolchain
- **Ejercicios**: Problemas de configuración y troubleshooting

## Herramientas

- **Git 2.40+**
- **CMake 3.20+**
- **VS Code** con extensiones (C/C++, CMake Tools, GitLens)
- **GDB/LLDB** para depuración
- **Valgrind**, **AddressSanitizer**, **UndefinedBehaviorSanitizer**
- **GitHub/GitLab** para hosting de repositorios
