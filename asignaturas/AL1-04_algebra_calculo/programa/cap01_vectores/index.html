    Idea Clave¶

    Una se√±al de audio digital es una secuencia ordenada de n√∫meros (muestras). Podemos representarla como un vector y aplicar operaciones algebraicas para calcular m√©tricas, proyecciones y transformaciones.

    Vector como Se√±al¶

    Definici√≥n¶

    Un vector en \(\mathbb{R}^n\) es una tupla ordenada de \(n\) n√∫meros reales:

    \[ \vec{x} = \begin{bmatrix} x_1 \\ x_2 \\ \vdots \\ x_n \end{bmatrix} \]

    Interpretaci√≥n en Audio¶

    Para una se√±al de audio con \(N\) muestras a frecuencia de muestreo \(f_s\):

    • \(x[n]\) es la amplitud de la muestra \(n\)
    • \(n = 0, 1, 2, \ldots, N-1\) (√≠ndices discretos)
    • Tiempo: \(t_n = n / f_s\)

    Ejemplo: Sinusoide de 1kHz a 48kHz durante 10ms

    \[ x[n] = A \sin(2\pi f \cdot n / f_s), \quad n = 0, 1, \ldots, 479 \]

    donde \(N = 0.01 \times 48000 = 480\) muestras.

    Operaciones B√°sicas¶

    Suma de Vectores¶

    \[ \vec{x} + \vec{y} = \begin{bmatrix} x_1 + y_1 \\ x_2 + y_2 \\ \vdots \\ x_n + y_n \end{bmatrix} \]

    Aplicaci√≥n: Mezcla de dos se√±ales mono.

    Multiplicaci√≥n por Escalar¶

    \[ \alpha \vec{x} = \begin{bmatrix} \alpha x_1 \\ \alpha x_2 \\ \vdots \\ \alpha x_n \end{bmatrix} \]

    Aplicaci√≥n: Ganancia o atenuaci√≥n (\(\alpha = 0.5\) ‚Üí -6dB aprox).

    Ejemplo Num√©rico¶

    Sean dos buffers de 4 muestras:

    \[ \vec{a} = \begin{bmatrix} 0.5 \\ 0.8 \\ -0.3 \\ 0.1 \end{bmatrix}, \quad \vec{b} = \begin{bmatrix} 0.2 \\ -0.4 \\ 0.6 \\ 0.0 \end{bmatrix} \]

    Mezcla 50/50:

    \[ \vec{m} = 0.5\vec{a} + 0.5\vec{b} = \begin{bmatrix} 0.35 \\ 0.20 \\ 0.15 \\ 0.05 \end{bmatrix} \]

    Normas de Vectores¶

    Norma L1¶

    \[ \|\vec{x}\|_1 = \sum_{i=1}^{n} |x_i| \]

    Uso: Suma de amplitudes absolutas (relacionada con energ√≠a perceptual).

    Norma L2 (Euclidiana)¶

    \[ \|\vec{x}\|_2 = \sqrt{\sum_{i=1}^{n} x_i^2} \]

    Uso: Energ√≠a de la se√±al, base del c√°lculo de RMS.

    Relaci√≥n con RMS¶

    El RMS (Root Mean Square) normaliza la norma L2 por el n√∫mero de muestras:

    \[ \text{RMS} = \frac{\|\vec{x}\|_2}{\sqrt{N}} = \sqrt{\frac{1}{N} \sum_{i=1}^{N} x_i^2} \]

    Ejemplo: Buffer de 4 muestras \(\vec{x} = [0.5, -0.5, 0.5, -0.5]\)

    \[ \|\vec{x}\|_2 = \sqrt{0.25 + 0.25 + 0.25 + 0.25} = 1.0 \]
    \[ \text{RMS} = \frac{1.0}{\sqrt{4}} = 0.5 \]

    En dBFS (asumiendo fullscale = 1.0):

    \[ \text{dBFS} = 20 \log_{10}(0.5) \approx -6.02 \text{ dB} \]

    Producto Escalar¶

    Definici√≥n¶

    \[ \vec{x} \cdot \vec{y} = \sum_{i=1}^{n} x_i y_i = x_1 y_1 + x_2 y_2 + \cdots + x_n y_n \]

    Propiedades¶

    • Conmutativo: \(\vec{x} \cdot \vec{y} = \vec{y} \cdot \vec{x}\)
    • Distributivo: \(\vec{x} \cdot (\vec{y} + \vec{z}) = \vec{x} \cdot \vec{y} + \vec{x} \cdot \vec{z}\)
    • Relaci√≥n con norma: \(\vec{x} \cdot \vec{x} = \|\vec{x}\|_2^2\)

    Interpretaci√≥n Geom√©trica¶

    \[ \vec{x} \cdot \vec{y} = \|\vec{x}\|_2 \|\vec{y}\|_2 \cos(\theta) \]

    donde \(\theta\) es el √°ngulo entre los vectores.

    Ortogonalidad: \(\vec{x} \perp \vec{y} \iff \vec{x} \cdot \vec{y} = 0\)

    Aplicaci√≥n: Correlaci√≥n¶

    La correlaci√≥n mide similitud entre se√±ales:

    \[ \text{corr}(\vec{x}, \vec{y}) = \frac{\vec{x} \cdot \vec{y}}{\|\vec{x}\|_2 \|\vec{y}\|_2} \]

    Rango: \([-1, 1]\)

    • \(+1\): Se√±ales id√©nticas (en fase)
    • \(0\): No correlacionadas (ortogonales)
    • \(-1\): Antifase completa

    Ejemplo: Detecci√≥n de fase entre L y R

    import numpy as np

L = np.array([0.5, 0.8, 0.3, -0.2])
R = np.array([0.5, 0.8, 0.3, -0.2])  # Id√©ntico

corr = np.dot(L, R) / (np.linalg.norm(L) * np.linalg.norm(R))
print(f"Correlaci√≥n: {corr:.3f}")  # 1.000

    Proyecci√≥n Ortogonal¶

    Problema¶

    Dado un vector \(\vec{x}\) y una direcci√≥n unitaria \(\hat{u}\), encontrar la componente de \(\vec{x}\) en direcci√≥n \(\hat{u}\).

    F√≥rmula¶

    La proyecci√≥n escalar de \(\vec{x}\) sobre \(\hat{u}\):

    \[ \text{proj}_{\hat{u}}(\vec{x}) = \vec{x} \cdot \hat{u} \]

    El vector proyecci√≥n:

    \[ \vec{p} = (\vec{x} \cdot \hat{u}) \hat{u} \]

    Ejemplo: Se√±al sobre Coseno¶

    Sea \(\vec{x}\) un buffer y \(\hat{u}\) un coseno normalizado:

    \[ u[n] = \cos(2\pi f n / f_s), \quad \hat{u} = \frac{\vec{u}}{\|\vec{u}\|_2} \]

    La proyecci√≥n \(\vec{x} \cdot \hat{u}\) da la amplitud de la componente frecuencial \(f\) en \(\vec{x}\) (base de la DFT).

    Micro-laboratorio¶

    Objetivo¶

    Calcular RMS y correlaci√≥n entre canales L/R de un archivo est√©reo sint√©tico.

    Setup¶

    import numpy as np

# Generar se√±al est√©reo: L = tono, R = tono con ruido
fs = 48000
duration = 0.1  # 100ms
t = np.arange(int(fs * duration)) / fs

# Canal L: sinusoide pura 1kHz
L = 0.5 * np.sin(2 * np.pi * 1000 * t)

# Canal R: mismo tono + ruido
np.random.seed(42)
R = 0.5 * np.sin(2 * np.pi * 1000 * t) + 0.1 * np.random.randn(len(t))

    Tareas¶

    1. Calcular RMS de cada canal
    rms_L = np.sqrt(np.mean(L**2))
rms_R = np.sqrt(np.mean(R**2))

print(f"RMS L: {rms_L:.4f}")
print(f"RMS R: {rms_R:.4f}")
    1. Calcular correlaci√≥n L-R
    corr = np.dot(L, R) / (np.linalg.norm(L) * np.linalg.norm(R))
print(f"Correlaci√≥n L-R: {corr:.4f}")

    Preguntas¶

    • ¬øPor qu√© el RMS de R es mayor que el de L?
    • ¬øQu√© indica una correlaci√≥n cercana a 1?
    • ¬øC√≥mo afectar√≠a invertir la fase de R?

    C√≥digo Completo¶

    import numpy as np

duration = 0.1
t = np.arange(int(fs * duration)) / fs

rms_R = np.sqrt(np.mean(R**2))
corr = np.dot(L, R) / (np.linalg.norm(L) * np.linalg.norm(R))

print(f"RMS L: {rms_L:.4f}")
print(f"RMS R: {rms_R:.4f}")
print(f"Correlaci√≥n: {corr:.4f}")

RMS R: 0.3571

    Ejercicios¶

    Ejercicio 1: C√°lculo de RMS¶

    Dado el buffer \(\vec{x} = [0.8, -0.6, 0.4, -0.2]\):

    a) Calcula \(\|\vec{x}\|_2\) b) Calcula el RMS c) Convierte a dBFS

    Soluci√≥n:

    a) \(\|\vec{x}\|_2 = \sqrt{0.64 + 0.36 + 0.16 + 0.04} = \sqrt{1.2} \approx 1.095\)

    b) \(\text{RMS} = 1.095 / \sqrt{4} = 0.548\)

    c) \(\text{dBFS} = 20 \log_{10}(0.548) \approx -5.22\) dB

    Ejercicio 2: Ortogonalidad¶

    Verifica si los vectores son ortogonales:

    \[ \vec{a} = [1, 0, -1], \quad \vec{b} = [1, \sqrt{2}, 1] \]

    Soluci√≥n:

    \[ \vec{a} \cdot \vec{b} = 1 \cdot 1 + 0 \cdot \sqrt{2} + (-1) \cdot 1 = 0 \]

    S√≠ son ortogonales.

    Ejercicio 3: Proyecci√≥n¶

    Proyecta \(\vec{x} = [3, 4]\) sobre \(\vec{u} = [1, 0]\).

    Soluci√≥n:

    \(\vec{u}\) ya es unitario (\(\|\vec{u}\|_2 = 1\)).

    \[ \text{proj}_{\vec{u}}(\vec{x}) = \vec{x} \cdot \vec{u} = 3 \]

    Vector proyecci√≥n: \(\vec{p} = 3 \cdot [1, 0] = [3, 0]\)

    Ejercicio 4: Implementaci√≥n¶

    Escribe una funci√≥n Python que calcule la correlaci√≥n entre dos arrays de igual longitud.

    Soluci√≥n:

    def correlacion(x, y):
    dot = np.dot(x, y)
    norm_x = np.linalg.norm(x)
    norm_y = np.linalg.norm(y)
    return dot / (norm_x * norm_y)

# Ejemplo
a = np.array([1, 2, 3])
b = np.array([2, 4, 6])
print(correlacion(a, b))  # 1.0 (proporcionales)
