    Conceptos Previos¶

    Transformada de Fourier¶

    La FFT (Fast Fourier Transform) descompone una se√±al temporal en sus componentes frecuenciales:

    \[ X(k) = \sum_{n=0}^{N-1} x[n] e^{-j 2\pi kn/N} \]

    Donde:

    • \(x[n]\): Se√±al en tiempo
    • \(X(k)\): Coeficientes de frecuencia (bins)
    • \(k\): √çndice de bin (frecuencia)
    • \(N\): Tama√±o de ventana FFT

    Ventanas¶

    Para reducir leakage espectral, aplicamos ventanas:

    • Hann: Buena para se√±ales tonales
    • Hamming: Similar a Hann, menos leakage
    • Blackman: M√°xima supresi√≥n de l√≥bulos laterales
    • Rectangular: Sin ventana (solo para se√±ales peri√≥dicas exactas)

    Integraci√≥n con Python¶

    Script B√°sico¶

    import numpy as np
import matplotlib.pyplot as plt
from scipy.fft import fft, fftfreq
from scipy.signal import hann
import soundfile as sf

# Cargar audio
audio, fs = sf.read("mi_audio.wav")

# Configuraci√≥n FFT
N = 4096
window = hann(N)

# Tomar segmento y aplicar ventana
segment = audio[:N] * window

# Calcular FFT
spectrum = fft(segment)
freqs = fftfreq(N, 1/fs)

# Magnitud en dB
magnitude_db = 20 * np.log10(np.abs(spectrum[:N//2]))

# Plotear
plt.figure(figsize=(12, 6))
plt.plot(freqs[:N//2], magnitude_db)
plt.xlabel("Frecuencia (Hz)")
plt.ylabel("Magnitud (dB)")
plt.title("Espectro de Frecuencia")
plt.grid(True)
plt.xlim(20, 20000)
plt.xscale('log')
plt.savefig("espectro_python.png")
plt.show()

    Identificaci√≥n de Arm√≥nicos¶

    Para un tono puro de frecuencia \(f_0\), los arm√≥nicos aparecen en:

    \[ f_n = n \cdot f_0, \quad n = 1, 2, 3, \ldots \]

    Ejemplo: Tono 100 Hz

    • Fundamental: 100 Hz
    • 2¬∫ arm√≥nico: 200 Hz
    • 3¬∫ arm√≥nico: 300 Hz

    Ancho de Banda¶

    Mide la frecuencia de corte (-3 dB) para filtros:

    \[ f_c = \text{frecuencia donde magnitud cae 3 dB respecto al pico} \]

    THD (Distorsi√≥n Arm√≥nica Total)¶

    Suma de energ√≠a de arm√≥nicos respecto a fundamental:

    \[ \text{THD} = \frac{\sqrt{P_2^2 + P_3^2 + \cdots}}{P_1} \times 100\% \]

    Donde \(P_n\) es la potencia del \(n\)-√©simo arm√≥nico.
